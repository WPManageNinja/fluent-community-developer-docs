# API Changelog

Track changes, updates, and new features in the Fluent Community REST API.

## Version 1.0.0 (Current)

### Initial Release

The first stable release of the Fluent Community REST API.

#### Features

- **Posts API**: Full CRUD operations for community posts
- **Comments API**: Create, read, update, and delete comments
- **Users API**: User management and profiles
- **Spaces API**: Community spaces and groups management
- **Reactions API**: Like and emoji reactions
- **Media API**: File upload and management
- **Notifications API**: User notification system
- **Bookmarks API**: Save and manage bookmarked posts
- **Feeds API**: Activity feeds and timelines
- **Search API**: Search across all content
- **Settings API**: Community configuration

#### Authentication

- WordPress Cookie Authentication
- Application Passwords
- JWT Token Support
- OAuth 2.0 Support

#### Features

- Pagination support for all list endpoints
- Filtering and sorting capabilities
- Include related data with `include` parameter
- Consistent error response format
- Rate limiting
- Comprehensive documentation
